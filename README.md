A professional, animated single-page website for **AfricaInUs**, showcasing:

- **Flyers & Print Design**

## Stack

- **Next.js 14** (App Router)
- **TypeScript**
- **Tailwind CSS**
- **Framer Motion** (animations)

## Run locally

```bash
cd africainus-website
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Build

```bash
npm run build
npm start
```

## Customization

- **Brand colors**: Edit `tailwind.config.ts` (primary, accent, dark).
- **Content**: Update copy in `src/components/` (Hero, Services, About, etc.).
- **Contact**: Replace the Contact form submit handler with your backend or email service.
